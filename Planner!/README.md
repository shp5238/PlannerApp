# 🧠✨ Custom Planner App (Open Source)

A feature-rich, customizable productivity planner built for people who want more than the limits of generic planning apps.
This app works for you and is designed to give you the best features to maximize your productivity. It is intended for self-starters and hard workers. 
With gamification, offline support, and beautiful personalization options, this open-source starter kit helps you build the planner you've always wanted—from minimal to maximal, from Pomodoro-focused to calendar-driven.

---

## 🚀 Features

### 📅 Core Tools
- **Calendar View** – Seamlessly manage your schedule.
- **To-Do List** – With integrated **Pomodoro Mode**.
  - Optional music sync: **Spotify**, **YouTube**, or **Apple Music**.
- **Notepad** – Quick notes and reflections, distraction-free.

### 🎮 Gamification
- **Daily login rewards** – 5 points per day to build strong morning habits.
- **Streak tracker** – Encourages consistency.
- **Rest day power-ups** – Earned by completing challenges: *Achievement card unlocked!*
- **Coins system** – Buy templates, fonts, timer sounds, background elements.
- **Custom Alerts** – Time-based or location-based. Ex: “At library? Time to study!”

### 🎨 Aesthetic Customization
- **Themes** – Examples: `Minecraft`, `Barbie` (more to come!)
- **App Modes** – Toggle between **Minimalist** or **Maximalist** view.
- **Custom fonts**, sounds, backgrounds, and templates.

### 🧘‍♀️ Motivation + Wellness
- **Morning 5-Minute Journal Challenge** – Optional daily check-in.
- **Motivational Quote Banners** – Auto-refreshing inspiration.
- **Progress Status Bar** – Visualize your daily wins.

### 🔗 Integrations
- **Google Calendar**, **Apple Calendar**, **Outlook** sync support.
- **Offline support** – Full functionality even without internet.
- **Optional AI Assistant** – Suggests prioritization & schedules your day automatically.

## 🛠️ Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/custom-planner-app.git

```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Development Server
```bash
npm run dev
```
